# 📈 AlphaCheck: 智慧型美股買賣決策系統

這是一個基於 Python 開發的 Web 應用程式，旨在幫助投資者透過數據分析做出更客觀的買賣決策。

## 🚀 核心功能
- **即時數據抓取**：利用 `yfinance` 獲取美股最新報價。
- **三指標打分邏輯**：
  1. **趨勢動能**：股價是否站穩 50 日均線。
  2. **估值分析**：本益比 (P/E Ratio) 是否處於合理區間。
  3. **分析師評級**：結合華爾街分析師的共識建議。
- **自動決策建議**：系統會給出「強力買入」、「觀望」或「避開」的結論。

## 🛠️ 技術棧
- **語言**：Python
- **網頁框架**：Streamlit
- **數據分析**：Pandas, Plotly
- **部署平台**：GitHub + Streamlit Cloud

## 📂 檔案說明

- `requirements.txt`: 專案所需的套件清單。
